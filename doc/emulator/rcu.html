Functions

love.load ()
An event-listener that starts the onStart() function An event-listener that starts the user defined onStart() function. Runs when the application is loaded. In the case of the emulator, the onStart() function needs to be defined if the RCU module is required otherwise the application crashes.
love.keypressed (key, isrepeat)
Detects when a key is pressed A function that detects when a key is pressed. Determines the state of the key and passes it to the user defined onKey() function. Uses the game engine Love2D to realize the function.

Parameters:

  • key the key pressed
  • isrepeat boolean stating if the key has entered the 'up' state since it last was 'down'
love.keyreleased (key)
Detects when a key is released A function that detects when a key is released. Passes the released key to the user defined onKey() function. Uses the game engine Love2D to realize the function.

Parameters:

  • key the key pressed
