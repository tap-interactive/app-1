parse_text_to_surface (text, letter_width, letter_height, max_width, max_height)
The function that processes the supplied text and divides it into characters. The function that processes the supplied text and divides it into characters. The characters are converted to ASCII-numbers which are then passed to the get_chr_img-function. This function also handles the format of the printed text, if the next word does not fit the rest of the line the word is printed on the next line. Words that are too long to fit a line are divided and written over two lines.

Parameters:

  • text a string containing the text to be printed on the text surface
  • letter_width the width of each letter in pixels
  • letter_height the height of each letter in pixels
  • max_width the maximum width of the text surface, in pixels
  • max_height the maximum height of the text surface, in pixels
print_text (mixed_text, x, y, max_width, max_height, font)
Prints text on the screen. Prints a given text on a transparent surface which is placed on the screen. The surface size is dependent on the length of the string as well the restrictions of allowed maximum width and height.

Parameters:

  • mixed_text a string containing the text to be printed
  • x the x-position of the text surface's upper left corner on the screen
  • y the y-position of the text surface's upper left corner on the screen
  • max_width the allowed width of the text surface in pixels
  • max_height the allowed height of the text surface in pixels
  • font a string with the name of the font that shall be used to print the text
