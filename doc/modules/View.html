view.set_gameboard_sizes (no_of_players)
Calculates player image sizes and positions Calculates player image sizes and positions using the number of players and the game board size

Parameters:

  • no_of_players the number of players chosen for the game
view.display_question (this_question)
Shows the questions and corresponding answer alternatives Start by setting a light grey background on the upper part of the screen, the function will then load a picture for the colour buttons and then print text for title, question and alternatives.The bottom bar will be cleaned to only contain the exit-button.

Parameters:

  • this_question the question to be displayed
view.display_correct_answer (current_correct_answer, current_category)
Display the correct answer on the screen The area where the answer alternatives are displayed is cleaned, and then only the correct answer is displayed with a green border on the color dot. An advertisement is displayed next to the correct answer, matching the category of the current question. In the bottom black bar a help text is displayed.

Parameters:

  • current_correct_answer is the the correct answer of the current question and should be shown
  • current_category the category of the current question
view.update_game_board ()
Updates the game board Cleans the game board area and displays goal, tracks and players and bananas (if game mode is set to "Go bananas!")
view.display_players ()
Displays the players character images. For each player, display the corresponding character image at start. Calculate image y-position using image size and number of players.
view.display_track ()
A function that prints the racing tracks for the players. A function that prints the racing tracks for the players. It is adjusted according to the images size of the players and the track belonging to the current player is highlighted by being filled with another color than the other tracks.
view.display_numbers ()
Displays numbers on the tracks Displays images as numbers 1-6 in the beginning of the tracks. The images are scaled to fit the size of the tracks, which depends on the number of players.
view.display_banana_peel ()
A function that prints banana peels on the racing track. Displays an image as banana peels for each player. The number of banana peels on each track is calculated by dividing the scores to win by 5 and are randomly placed on the track.
view.display_goal ()
Display goal line Displays an image as goal line
view.clean_game_board ()
Function that cleans the game board surface Cleans the game board surface, e.g. the surface where the tracks and player charachters are displayed. The surface is filled with the same a light grey color, which is used as the background color in the game mode.
view.display_win ()
Displays the medal winners at the end of the game. Shows a pop-up with character images of the player(s) that won or got the second or third place in the game. The positions of the character images is calculated according to number of winners, since there can be 1-6 players at each placement.
view.display_image (url, img_size, img_pos)
Help function to display an image on the screen Help function to display an image on the screen given the image path, the size of the displayed image and start position (upper left corner) of where to display the image. If the image dimensions is larger than the the img_size param the image will be scaled to the dimensions of img_size

Parameters:

  • url image path
  • img_size size of the displayed image on the format {w=img_width, h=img_height}
  • img_pos position on the screen where to display the image on the format {x=x_pos, y=y_pos}
